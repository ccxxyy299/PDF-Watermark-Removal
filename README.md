# PDF-Watermark-Removal

PDF 去水印工具，支持去除半透明灰色水印，通过 Web 界面操作。

## 安装依赖

```bash
pip install flask opencv-python numpy pymupdf Pillow
```

## 运行

```bash
python app.py
```

浏览器打开 `http://127.0.0.1:5000`，上传 PDF 文件，调节灵敏度后点击去除水印，下载处理后的文件。

## 原理

1. **PDF 转图片**：使用 PyMuPDF 将 PDF 逐页转为 400DPI 的 PNG 图片
2. **水印检测**：通过 LAB 和 HSV 色彩空间识别水印
   - 水印特征：高亮度（L=220-245）+ 极低饱和度（S<4）
3. **连通区域过滤**：去除噪点，保留大面积水印区域
4. **边缘保护**：使用 Canny 边缘检测保护有清晰轮廓的内容（文字、图片、人像）
5. **图片转 PDF**：处理后的图片合并为新 PDF

## 灵敏度说明

| 值 | 效果 |
|---|------|
| 20-40 | 保守模式，只去除明显水印，保护内容 |
| 40-60 | 平衡模式（默认） |
| 60-80 | 激进模式，去除更彻底，可能影响灰色内容 |

## 注意事项

- 主要针对**半透明灰色水印**，对彩色或深色水印效果有限
- 处理时间约 0.5-1 秒/页，页数多时请耐心等待
- 灰色衣服、灰色背景等内容可能被误伤，建议降低灵敏度
- 转图片再转回 PDF 会有轻微质量损失
