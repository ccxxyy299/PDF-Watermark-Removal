# PDF-Watermark-Removal

PDF 去水印工具，支持多种类型水印去除，通过 Web 界面操作。

## 安装依赖

```bash
pip install flask opencv-python numpy pymupdf Pillow
```

## 运行

```bash
python app.py
```

浏览器打开 `http://127.0.0.1:5000`，上传 PDF 文件，选择水印类型和灵敏度后点击处理，下载处理后的文件。

## 支持的水印类型

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 自动检测 | 综合策略，自适应识别水印 | 通用场景（推荐） |
| 灰色水印 | 半透明/浅灰水印 | "CONFIDENTIAL"等灰色水印 |
| 彩色水印 | 有颜色的水印 | 红色/蓝色等彩色水印 |
| 深色水印 | 深色背景水印 | "DRAFT"等深色水印 |

## 去水印原理

采用双层处理策略：

1. **PDF原生处理**：解析PDF content stream，识别并移除半透明水印指令（旋转文字、透明度设置），从源头去除水印
2. **图像级处理**：PDF转高分辨率图片后，通过色彩空间分析（LAB/HSV）检测水印像素，结合边缘保护算法确保内容完整
3. **合成输出**：处理后的图片重新合并为PDF

## 灵敏度说明

| 值 | 效果 |
|---|------|
| 20-40 | 保守模式，只去除明显水印，保护内容 |
| 40-70 | 平衡模式（默认60） |
| 70-100 | 激进模式，去除更彻底，可能影响灰色内容 |

## 注意事项

- 支持灰色、彩色、深色等多种水印类型
- 处理时间约 0.5-1 秒/页
- 灰色内容（如灰色背景）在高灵敏度下可能被误伤
- PDF转图片再转回会有轻微质量损失
